## 프로젝트 소개

- KOSTA 284기 프론트엔드 주제로 약 3주간 프론트엔드 기술을 공부하여 진행한 팀플 과제 결과물 입니다.
- 기존의 숙박업소의 틀을 깨는 신비롭고 색다른 경험을 제공하는 게스트하우스 사이트입니다.
- 처음 접속하였을 때 보이는 메인 페이지와 특정 숙소를 클릭하였을 때 나오는 상세 페이지로 구성되어 있습니다.
